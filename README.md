# Dergcave
WIP clicker game made in Godot

You pet the dragon as he roams around his cave, using the affection gained to get gold, gems, mushrooms etc. to grow the hoard.
The numbers don't get to insane numbers like in cookie clicker, but there is still quite a lot of hours to be spent with this dragon.

Content is roughly 90% finished. Missing some art for upgrades and sprites. Left that for last, because I wanted to prioritize getting the game working first. 
Attached also a working build of the game, not completely tested, so late game and some upgrades might be broken, but early game will work 100%.
